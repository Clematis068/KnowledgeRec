# 2026-03-12 工作记录

## 今日改动

- 推荐链路新增登录上下文加权：
  - 登录地区 `last_login_region`
  - 登录时区 `last_login_timezone`
  - 登录时段 `last_login_time_slot`
- 推荐结果新增 `context_score`，并根据地区 / 时段匹配给轻量 bonus：
  - `REGION_MATCH_BONUS = 0.08`
  - `TIME_SLOT_MATCH_BONUS = 0.05`
  - `REGION_TIME_COMBO_BONUS = 0.03`
  - `CONTEXT_BONUS_CAP = 0.12`
- 帖子新增上下文字段：
  - `target_regions`
  - `target_time_slots`
- 登录与推荐请求统一从前端自动附带浏览器上下文：
  - `X-Client-Timezone`
  - `X-Client-Time-Slot`
- 地区来源从前端 locale 推断调整为后端优先读取代理 / CDN 注入的国家码头，避免把界面语言误判为真实地区
- 发帖 / 编辑页支持设置内容适配地区与适配时间段

## 补充说明

- 上下文推荐采用轻量加分，不直接替代用户兴趣与交互行为。
- 对历史帖子做了温和回退：
  - 时间段未配置时，按帖子发布时间推断
  - 地区未配置时，优先尝试作者最近登录地区
