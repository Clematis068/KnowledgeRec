# 09 — 服务层

## 千问服务 (`qwen_service.py`)

**类**: `QwenService` (全局单例 `qwen_service`)

**连接方式**: OpenAI SDK 兼容接口
```python
client = OpenAI(
    api_key=DASHSCOPE_API_KEY,
    base_url='https://dashscope.aliyuncs.com/compatible-mode/v1'
)
```

### 方法列表

| 方法 | 用途 | 模型 |
|------|------|------|
| `chat(message, system_prompt)` | 单轮对话 | qwen-plus |
| `chat_with_history(messages)` | 多轮对话 | qwen-plus |
| `get_embedding(text)` | 文本向量化 | text-embedding-v3 |
| `extract_tags(content)` | 提取标签/摘要/领域 | qwen-plus |

### `extract_tags` Prompt
```
请分析文章内容，返回JSON: {"tags": [3-5个标签], "summary": "一句话摘要", "domain": "所属领域"}
```

### Embedding 输出
- 模型: `text-embedding-v3`
- 维度: 1024
- 返回: `list[float]`

## Neo4j 服务 (`neo4j_service.py`)

**类**: `Neo4jService` (全局单例 `neo4j_service`)

**连接**: `neo4j.GraphDatabase.driver(bolt://localhost:7687)`

### 方法

| 方法 | 用途 |
|------|------|
| `run_query(cypher, params)` | 只读查询，返回 `list[dict]` |
| `run_write(cypher, params)` | 写操作 (CREATE/MERGE) |
| `close()` | 关闭连接 |

**特点**: 懒初始化，首次调用时才创建 driver

**主要使用者**: `graph_engine.py` — 执行 3跳涟漪传播 Cypher 查询

---

## Redis 服务 (`redis_service.py`)

**类**: `RedisService` (全局单例 `redis_service`)

**连接**: `redis://localhost:6379/0`

### 方法

| 方法 | 用途 | 默认 TTL |
|------|------|----------|
| `set_json(key, value, ttl)` | 存储 JSON | 3600s (1h) |
| `get_json(key)` | 读取 JSON | — |
| `set_sorted_set(key, mapping, ttl)` | 存有序集合 | 86400s (24h) |
| `get_top_from_sorted_set(key, count)` | 取 Top-N | — |

### 缓存 Key 设计

| Key 模式 | 数据类型 | 内容 | TTL |
|----------|----------|------|-----|
| `item_sim:{post_id}` | Sorted Set | Top-50 相似物品 {member: score} | 24h |
| `llm_rel:{user_id}:{post_id}` | JSON | LLM 相关性评分 (0~1) | 1h |
