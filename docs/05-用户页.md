# 05 — 用户页

## UserCard.vue（用户卡片）
- **功能**：用户预览卡片，点击跳转用户详情
- **展示**：
  - 头像 (`el-avatar`，取用户名首字母，蓝底)
  - 用户名 + 简介
  - 兴趣画像标签
- **交互**：整卡点击 → `/users/${user.id}`

## UserListPage.vue（用户列表页）
- **布局**：响应式网格 (`el-row` + `el-col`)
  - xs=24（手机1列）, sm=12（2列）, md=8（3列）, lg=6（4列）
- **分页**：`el-pagination`，每页 20 条
- **API**：`GET /api/user/list?page=1&per_page=20`

## UserDetailPage.vue（用户详情页）
- **布局**：
  - 返回按钮 (`el-page-header`)
  - 个人资料卡片：头像 + 用户名 + 邮箱 + 简介 + 兴趣画像 + 注册时间
  - 行为历史卡片：`BehaviorTimeline` 组件
- **API**：并行请求 `getUserDetail` + `getUserBehaviors`

## BehaviorTimeline.vue（行为时间线）
- **功能**：`el-timeline` 展示用户历史行为
- **行为类型映射**：
  | 类型 | 标签 | 颜色 |
  |------|------|------|
  | view | 浏览 | primary (蓝) |
  | like | 点赞 | success (绿) |
  | comment | 评论 | warning (橙) |
  | share | 分享 | danger (红) |
- **每项展示**：行为类型 + 帖子链接 + 持续时间(秒)
