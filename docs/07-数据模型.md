# 07 — 数据模型

## ER 关系图
```
Domain 1──N Tag
  │           │
  │ 1──N      M──N
  │           │
  Post M──N Tag (post_tag 关联表)
  │
  N──1 User (author)
  │
  1──N UserBehavior N──1 User

  User N──N User (UserFollow 自关联)
```

## MySQL 模型

### User
```python
字段:
  id            INT PK
  username      VARCHAR(64) UNIQUE
  email         VARCHAR(128) UNIQUE
  avatar_url    VARCHAR(256)
  bio           TEXT
  interest_embedding  JSON    # 1024维向量，加权平均(liked/favorited posts)
  interest_profile    TEXT    # LLM 生成的兴趣画像描述
  created_at    DATETIME
  updated_at    DATETIME

关系:
  posts      → Post (1:N, 作者)
  behaviors  → UserBehavior (1:N)
```

### Post
```python
字段:
  id            INT PK
  title         VARCHAR(256) NOT NULL
  content       TEXT NOT NULL
  summary       VARCHAR(512)     # LLM 提取的摘要
  author_id     INT FK→User
  domain_id     INT FK→Domain
  view_count    INT DEFAULT 0
  like_count    INT DEFAULT 0
  content_embedding  JSON        # 1024维向量
  created_at    DATETIME
  updated_at    DATETIME

关系:
  tags       → Tag (M:N, via post_tag)
  author     → User (N:1)
  domain     → Domain (N:1)
  behaviors  → UserBehavior (1:N)
```

### Tag
```python
字段: id, name(UNIQUE), domain_id(FK), embedding(JSON 1024维)
关系: domain → Domain, posts → Post (M:N)
```

### Domain
```python
字段: id, name(UNIQUE), description
关系: tags → Tag (1:N), posts → Post (1:N)
```

### UserBehavior
```python
字段:
  id              INT PK
  user_id         INT FK (indexed)
  post_id         INT FK (indexed)
  behavior_type   ENUM('browse', 'like', 'favorite', 'comment')
  comment_text    TEXT           # 仅 comment 类型
  duration        INT            # 浏览时长(秒)，仅 browse
  created_at      DATETIME (indexed)
```

### UserFollow
```python
字段: follower_id(PK, FK), followed_id(PK, FK), created_at
# 联合主键，自关联
```

## Neo4j 图谱 Schema

### 节点
| 节点类型 | 属性 | 来源 |
|---------|------|------|
| User | id, username | MySQL.User |
| Post | id, title | MySQL.Post |
| Tag | id, name | MySQL.Tag |
| Domain | id, name | MySQL.Domain |

### 关系
| 关系 | 方向 | 说明 |
|------|------|------|
| AUTHORED | User → Post | 作者关系 |
| LIKED | User → Post | 点赞 |
| FAVORITED | User → Post | 收藏 |
| COMMENTED | User → Post | 评论 |
| BROWSED | User → Post | 浏览 |
| FOLLOWS | User → User | 关注 |
| TAGGED_WITH | Post → Tag | 帖子标签 |
| BELONGS_TO | Tag → Domain | 标签所属领域 |
| INTERESTED_IN | User → Tag | 派生关系，weight=交互次数 |

### 约束与索引
```cypher
CREATE CONSTRAINT FOR (u:User) REQUIRE u.id IS UNIQUE
CREATE CONSTRAINT FOR (p:Post) REQUIRE p.id IS UNIQUE
CREATE CONSTRAINT FOR (t:Tag) REQUIRE t.id IS UNIQUE
CREATE CONSTRAINT FOR (d:Domain) REQUIRE d.id IS UNIQUE
```
