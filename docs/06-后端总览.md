# 06 — 后端总览

## 技术栈
- **Flask 3.1** — Web 框架
- **SQLAlchemy** (flask-sqlalchemy) — MySQL ORM
- **Neo4j 6** — 图数据库（知识图谱 + 涟漪传播）
- **Redis 7** — 缓存（物品相似度矩阵、LLM 评分）
- **千问 API** (DashScope OpenAI 兼容接口) — LLM 对话 + Embedding
- **uv** — 包管理（不用 pip）

| 千问模型 | `qwen-plus` |
| 千问 Embedding | `text-embedding-v3`（1024维） |

## Flask 工厂 (`__init__.py`)
```python
def create_app():
    app = Flask(__name__)
    CORS(app)
    db.init_app(app)
    # 注册 4 个 Blueprint
    app.register_blueprint(llm_bp, url_prefix='/api/llm')
    app.register_blueprint(rec_bp, url_prefix='/api')
    app.register_blueprint(user_bp, url_prefix='/api/user')
    app.register_blueprint(post_bp, url_prefix='/api/post')
```
