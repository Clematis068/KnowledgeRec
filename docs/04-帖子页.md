# 04 — 帖子页

## PostCard.vue（帖子卡片）
- **功能**：帖子预览卡片，点击跳转到帖子详情
- **展示内容**：
  - 标题 + 领域标签
  - 摘要（2行截断）
  - 浏览数、点赞数、标签（最多3个）
- **交互**：整张卡片可点击 → `$router.push('/posts/${post.id}')`

## PostListPage.vue（帖子列表页）
- **领域筛选**：`el-tabs` 切换领域（全部 / 计算机科学 / 数学 / 物理学 / 生物学 / 经济学）
- **分页**：`el-pagination`，每页 20 条
- **数据流**：
  ```
  切换 tab → onDomainChange() → page=1 → fetchPosts()
  翻页 → @current-change → fetchPosts()
  ```
- **API**：`GET /api/post/list?page=1&per_page=20&domain_id=1`

## PostDetailPage.vue（帖子详情页）
- **布局**：
  - 返回按钮 (`el-page-header`)
  - 标题 + 元信息（领域、作者ID、浏览数、点赞数、时间）
  - 标签区域
  - 摘要区域（灰底卡片）
  - 正文内容（`white-space: pre-wrap`）
- **API**：`GET /api/post/{postId}`
