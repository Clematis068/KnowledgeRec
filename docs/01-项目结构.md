# 01 — 项目结构

## 技术栈
- **Vue 3.5** + Composition API (`<script setup>`)
- **Vite 7** — 开发服务器 + 构建
- **Element Plus** — UI 组件库 (中文 locale `zhCn`)
- **Vue Router 4** — 前端路由 (history 模式)
- **Axios** — HTTP 请求

## 路由表
| 路径 | 页面 | 说明 |
|------|------|------|
| `/recommend` | RecommendPage | 首页，智能推荐 |
| `/posts` | PostListPage | 帖子列表 + 领域筛选 |
| `/posts/:id` | PostDetailPage | 帖子详情 |
| `/users` | UserListPage | 用户网格 |
| `/users/:id` | UserDetailPage | 用户画像 + 行为历史 |

所有路由嵌套在 `AppLayout` 下，共享顶栏 + 侧边栏。`/` 重定向到 `/recommend`。

## Vite 代理配置
```js
server: {
  port: 5173,
  proxy: { '/api': { target: 'http://localhost:5000', changeOrigin: true } }
}
```
前端 `/api/*` 请求全部代理到 Flask 后端 5000 端口。

## 关键设计决策
- **不用 Pinia**：无登录系统，各页面独立 fetch，不需全局状态管理
- **全量注册 Icons**：`for (const [key, component] of Object.entries(ElementPlusIconsVue)) app.component(key, component)`，方便模板中直接用 `<Star />` 等
- **API 超时 30s**：推荐 API 会串行跑三个 Pipeline + LLM，耗时较长
